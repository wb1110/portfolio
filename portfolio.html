Free to Be Nutrition: Mobile App

Cross-platform nutrition application to help women lead healthier lives.

  • User completes an assessment that calculates macronutrient and caloric goals based on the results of the assessment.
  • User can search for and add foods using the USDA's API.
  • User can scan a product's barcode with their phone's camera to query the USDA API for that specific item.
  • User can track their caloric and macronutrient daily intake based on the foods entered for that day.
  • User's data is saved using AsyncStorage, because for the first MVP they will not access the app from anything other than their phone.
  • User's state is managed using zustand.
